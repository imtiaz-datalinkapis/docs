# DataLink APIs Documentation

This is the documentation site for DataLink APIs — a data intelligence gateway for email validation, phone HLR, IP intelligence, and domain intelligence.

## Structure

- `introduction.mdx` — Product overview and data types
- `quickstart.mdx` — First API call in 5 minutes
- `authentication.mdx` — Bearer token auth
- `credits.mdx` — Credit system and pricing
- `rate-limits.mdx` — Rate limits by plan
- `errors.mdx` — Error codes and format
- `api-reference/` — One page per endpoint

## Style

- Active voice, second person ("you")
- Sentence case for headings
- Bold for UI elements
- Code formatting for field names, endpoints, and commands
